# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 프로젝트 개요

이 저장소는 **"너구리의 모험"** 게임 개발을 위한 기획 문서 저장소입니다.

## 문서 구조

- `기획.md` - 게임 디자인 문서(GDD): 게임 컨셉, 시스템, 스테이지, 적 캐릭터, UI/UX, 수익화 모델
- `작업계획.md` - 8단계 개발 작업 계획서 (프리프로덕션 → 출시/라이브)

## 게임 정보

- **장르**: 2D 플랫폼 액션 게임
- **타겟 플랫폼**: PC, 모바일
- **월드 구성**: 봄의 숲, 여름 계곡, 가을 산, 겨울 동굴 (4개 월드, 각 5스테이지)

## 작업 시 참고사항

- 문서는 한국어로 작성
- 기획 변경 시 `기획.md` 수정 후 `작업계획.md`도 함께 업데이트 필요
- 작업계획의 체크박스(`- [ ]`)를 활용하여 진행 상황 추적
