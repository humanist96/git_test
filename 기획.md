# 너구리 게임 기획문서

## 1. 게임 개요

### 1.1 게임 컨셉
- **게임명**: 너구리의 모험
- **장르**: 2D 플랫폼 액션 게임
- **타겟 플랫폼**: PC, 모바일
- **타겟 유저**: 전연령

### 1.2 게임 소개
귀여운 너구리 캐릭터가 숲 속을 탐험하며 도토리를 모으고, 다양한 장애물을 피해 목표 지점에 도달하는 게임입니다.

---

## 2. 게임 시스템

### 2.1 핵심 메카닉
| 기능 | 설명 |
|------|------|
| 이동 | 좌우 이동, 점프, 이중 점프 |
| 변신 | 나뭇잎으로 변신하여 적을 속임 |
| 수집 | 도토리, 열매 등 아이템 수집 |
| 공격 | 꼬리 휘두르기로 적 공격 |

### 2.2 캐릭터 스탯
- **체력(HP)**: 기본 3칸
- **스태미나**: 달리기/변신 시 소모
- **점수**: 수집한 아이템으로 획득

### 2.3 아이템
| 아이템 | 효과 |
|--------|------|
| 도토리 | 점수 +10 |
| 황금 도토리 | 점수 +100 |
| 열매 | 체력 회복 +1 |
| 나뭇잎 | 변신 시간 증가 |

---

## 3. 스테이지 구성

### 3.1 월드 구성
1. **봄의 숲**: 튜토리얼 스테이지 (1-1 ~ 1-5)
2. **여름 계곡**: 물 장애물 등장 (2-1 ~ 2-5)
3. **가을 산**: 낙엽과 바람 기믹 (3-1 ~ 3-5)
4. **겨울 동굴**: 얼음 미끄러짐 (4-1 ~ 4-5)
5. **보스 스테이지**: 각 월드 마지막

### 3.2 난이도 설계
- 스테이지가 진행될수록 장애물 밀도 증가
- 제한 시간 도입 (중반부터)
- 히든 스테이지 존재
